###Bloccit: A Reddit replica!

**It has all the functionality of Reddit! Including voting, favoriting, commenting, creating posts and topics!**

####What's the Tech?
* Rails 4.2 + 4.1
* Ruby 2.0
* SQL3
* PRY
* Sendgrid

####Notable Gems:
* Figaro
* Devise
* New_Relic
* Capybara
* Factory_Girl 4.0
* Rspec 3.0
* Better_Errors 2.1
* Pundit
* JQuery
